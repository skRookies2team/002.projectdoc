
## ### 회의자료

### 1️⃣ 발표 분석 AI 서비스 기능 아이디어

#### 📂 PPT 업로드 기능
- 사용자가 **파워포인트 파일(PPT)** 업로드  
- 슬라이드별로 **표·텍스트 추출 및 내용 분석**  
- 기본 분석 목표:  
  - **서론–본론–결론 구조 검사**  
  - 또는 **헤드라인 분류 수준까지 구현**

#### 🎥 영상 병렬 처리
- 분석 처리 중, 사용자가 **직접 촬영한 발표 영상을 병렬 재생**  
- 분석 시간 동안 시각적 인터랙션 제공  
  → “AI가 분석하는 동안, 사용자는 본인 발표 확인 가능”

---

### 2️⃣ 음성 및 영상 생성 기능

#### 🗣️ 스크립트 → 음성 변환
- PPT 또는 영상에서 **발표 스크립트 추출 → TTS 변환**
- 결과물: **‘성우 버전 발표 영상’**  
  - 더 명확하고 또렷한 음성으로 재구성된 발표 영상  

#### 🎧 STT 활용 방식
- **실제 발표자의 음성 → 텍스트 대본화(STT)**  
- 실제 대본 vs 음성 기반 대본 비교 분석 가능  
- **TTS 음성 생성 시**  
  - 발표자의 음색을 학습시켜 **유사한 음성으로 변환 가능**  
  - 예: *Tortoise TTS* 등 활용 가능
